7Club - Mobile Club Management System

7Club is an Android application developed to streamline communication and data flow between university student clubs, their executive boards, and the central Clubs Union. The project focuses on real-time management and organizational transparency within a university ecosystem.

📋 Project Overview
Managing multiple student clubs requires efficient coordination. 7Club provides a centralized platform where:

Club Boards can manage internal tasks and member lists.

The Clubs Union can monitor club activities and approve requests.

Students can stay updated with club events and announcements.

✨ Key Features
Real-time Data Flow: Instant updates for announcements and event schedules.

Role-Based Access: Specific dashboards for Board Members, General Members, and the Clubs Union.

Management Tools: Tools for tracking club tasks and faculty approvals.

Team Collaboration: Developed as a collaborative project, emphasizing teamwork and professional coordination.
